## Getting Started ğŸš€

1. **Clone the Repository:**
    ```sh
    git clone https://github.com/asoral/nextproject.git
    ```

2. **Install Dependencies:**
    ```sh
    cd <frappe folder>
    bench get-app erpnext
    bench get-app Nextproject
    ```

    ```sh
    bench --site yoursite install-app nextproject
    ```

4. **Run the Server:**
    ```sh
    bench start
    ```
